# 📊 Портфоліо домашніх завдань: Data Analytics (Hillel IT School)

Ласкаво просимо до мого репозиторію з практичними роботами, виконаними під час навчання на курсі Data Analytics. Тут зібрано 29 практичних кейсів, які демонструють мої навички роботи з повним циклом даних: від збору та очищення до побудови інтерактивних дашбордів і розрахунку бізнес-метрик.

---

## 🛠️ Технологічний стек курсу
* **Python**: Pandas, NumPy, Matplotlib, Seaborn, Requests (API), Jupyter Notebooks
* **Бази даних**: PostgreSQL, складні SQL-запити (JOIN, оконні функції, CTE, PL/pgSQL)
* **BI-інструменти**: Power BI (Power Query, мова M, розрахунок мір DAX), Tableau (LOD-вирази)

---
